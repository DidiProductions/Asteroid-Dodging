# Asteroid-Dodging
Asteroid Dodging 
------------------------------------------------------------------------------------------------------------------------------------------
Asteroids are incoming.
1. Shoot with Spacebar 
2. Go up with w.
3. go down with s.
4. Pause with p.
There is a JoytoKey file you can use for your controller.
